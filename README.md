# Bin2Dec

Bin2Dec permite ao usuário inserir strings de até 8 dígitos binários, 0's
e 1's, em qualquer sequência e, em seguida, exibe seu equivalente decimal.

- Tecnologias utilizadas: ReactJS com TypeScript e Figma.
